## QA List

**Device performance check list**

  - iPhone X (375px / 2018)

- Tablet

  - iPad (768px / 8th generation - 2020)
  - iPad Pro (1024px / 2020)

- Desktop

  - MacBook Pro
  - Any Windows PC bigger than 1080px
